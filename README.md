# 배달 어플리케이션 아웃소싱 프로젝트 
   **프로젝트 기간 : 2025.01.07 ~ 2025.01.13**

배달 어플리케이션 개발에 대한 아웃소싱 프로젝트를 맡았다고 가정합니다.  
클라이언트는 배달 시장에 새롭게 진출하려는 스타트업으로, 
내부에 개발 인력이 부족한 상황에서 전체 개발 과정을 외주로 맡기기로 결정했습니다.  
기한은 단 일주일로, 기한을 넘기면 클라이언트에게 위약금을 물어주어야 합니다.

## 프로젝트 개요 

### 사용된 기술
* Java 17
* Spring boot: 3.4.1
  * Spring-validation
  * Spring-Data-Jpa
* Lombok
* MySQL
* BCrypt: 0.10.2
* JWT
* JUnit

### 주요 기능
- 회원 가입 (사장/고객)
- 로그인
- 회원 탈퇴
- 가게 CRUD
- 메뉴 CRUD
- 주문 CRUD
- 리뷰 CRUD
